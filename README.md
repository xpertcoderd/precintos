# Template Transfers 2.0

## Project setup
```
npm install
```
### Include the .env file (Archivo de variables de entorno):
```
VUE_APP_API = http://domainBackEnd.com
VUE_APP_HASH= YourToken
VUE_APP_SOCKET_SERVER= yourWebSocketServerIP
VUE_APP_API_KEY = YourGoogleAPiKey

```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
